# 《道渊》完整人物设定

---

## 主角 · 楚渊（唯一男主）

**核心设定：**
- 表面身份：青云城楚家旁支孤儿，父母早逝
- 真实身份：太古神魔族的遗孤，血脉被人为封印后被遗弃在低级世界
- 命格：九印之体——万古唯一能承载完整荒帝道渊的容器

**性格成长弧（跨1000章）：**
```
前期（1-150章）：隐忍、谨慎、以退为进
   → 中期（151-650章）：杀伐果断、重情重义、渐露锋芒
     → 后期（651-850章）：背负苍生、沉稳大气的领袖气质
       → 终局（851-1000+章）：俯瞰诸天的淡然与坚定
```

**关键转折点：**
- 第120章：第一次真正杀人
- 第350章：得知荒帝被背叛的真相
- 第630章：生父母出现
- 第680章：林霄战死
- 第830章：紫瑶为救他肉身陨落，他以第八印·时逆转，耗去三百年寿元
- 第920章：姬墨音战至道渊崩碎
- 第950章：直面荒帝背叛者

**核心矛盾：**
- 吞天诀越用越强，戾气越积越深
- 九印越完整，离"道渊尽头"越近，寿命上限越短
- 对紫瑶是宿命的牵引，对姬墨音是生死相托——但他知道终有一天要走道渊尽头，给不了任何人未来

## 女主一 · 紫瑶（清冷宿命型 · 感情线顺滑暧昧）

**定位：** 白月光、灵魂共鸣、自然的相知相惜

**设定：**
- 青云宗圣女，清冷绝尘，紫衣如烟，气质出尘
- 真实身份：荒帝直系后裔，体内封印着远古天凤血脉
- 姓"紫"，名"瑶"，单名即可入诗
- 与楚渊的相遇是荒帝道渊与荒帝血脉之间的宿命牵引——从第一眼对视开始，两人之间就有一根看不见的线

**暧昧感来源：**
紫瑶的性格清冷但不冷漠，她不会说肉麻的话，但每一个举动都透着在乎。暧昧感来自于"她都做到这个份上了，但她就是不说破"——楚渊也不敢问，怕一开口就打破了这种默契。

**与楚渊的感情线（顺滑自然，沿途撒糖）：**

| 阶段 | 章节 | 事件 |
|------|------|------|
| 初见 | 第18章 | 青云宗入门测试，两人对视，心跳同时漏了一拍。紫瑶面无表情地移开目光，但耳根红了 |
| 共处 | 第30章 | 宗门任务组队，七天七夜同行。紫瑶会在楚渊睡着时替他守夜，被发现后冷着脸说"只是顺手" |
| 暧昧期 | 第45章 | 月下练剑，紫瑶教他一招剑法，纠正姿势时指尖不经意碰到他的手背，两人同时僵住——然后她若无其事地收回去，但后半夜都没睡着 |
| 生死 | 第60章 | 秘境中被困绝境，她以为要死在那里了，靠在他肩上轻声说了一句"其实第一次见你的时候……我就知道你会是特别的人。"——然后推开他，补了一句"别多想，我只是陈述事实。" |
| 送别 | 第150章 | 第一卷末，楚渊离开青云域。紫瑶追到城外，塞给他一件用自己灵力温养了三年的护身玉佩，只说了一句"活着回来。"走出十步后，又补了一句"……我等你。" |
| 重逢 | 第250章 | 八荒排名大会上重逢，紫瑶已经是名震八荒的天之骄女。两人对视，她嘴角微微上扬了一下——那是全书她第一次笑 |
| 定情 | 第400章 | 天墟月下，两人并肩坐在万丈悬崖边。她靠在他肩上，轻声说"我一直都在等你来。"楚渊握住她的手，她没挣开 |
| 别离 | 第830章 | 为楚渊挡下致命一击，肉身陨落。陨落前她微笑着摸了摸他的脸——"别哭，我知道你会把我带回来的。" |
| 归来 | 第832章 | 楚渊以第八印·时逆转乾坤，耗费三百年寿元将她从死亡线上拉回。她醒来第一句话——"你老了。"然后笑了，笑中带泪 |
| 终章 | 第980章 | 她以天凤之火焚尽邪族大军，回头对楚渊笑——"这一次，换我来保护你。" |

**暧昧速写举例：**
> 紫瑶微微侧过头，目光落在楚渊的侧脸上。月光下，少年的轮廓比初见时硬朗了许多。她看了三息，然后若无其事地收回目光。
> 楚渊察觉到了，偏过头来看她："怎么了？"
> "没什么。"她顿了顿，"只是发现你好像长高了一点。"
> 楚渊愣了一下。他确实长高了——但那是在离开青云域之后的事，她怎么知道他之前多高？
> 她没解释，只是把目光投向远方的群山，嘴角有一个几乎不可察觉的弧度。

## 女主二 · 姬墨音（烈性火焰型 · 感情线曲折虐心）

**定位：** 烈火般的相遇，荆棘丛生的相知，以血与火淬炼出的深情

**设定：**
- 战皇殿圣女，八荒古族——姬家的继承人
- 姓"姬"，名"墨音"，古老尊贵的姓氏，承载着古族的荣耀与枷锁
- 性格：冷傲霸道、锋芒毕露、从不服软
- 一把黑色重剑"破军"，八千斤，一力破万法
- 她是楚渊遇到过最难搞的人——打不服、说不听、骂不走

**暧昧感来源：**
和紫瑶的"默契暧昧"完全不同，姬墨音的暧昧是"嘴里说着狠话，身体却无比诚实"。她嘴上说着"谁要管你的死活"，却在楚渊每次遇险时第一个冲到他身边。她从不承认自己在意他，但整个八荒都看得出来。

**与楚渊的感情线（曲折坎坷，步步荆棘）：**

| 阶段 | 章节 | 事件 |
|------|------|------|
| 冤家初遇 | 第180章 | 上古遗迹夺宝，两人大打出手，打了三天三夜不分胜负。最后她放狠话："下次见面，打断你的腿。" |
| 宿命对决 | 第250章 | 八荒排名大会，楚渊险胜半招。她当着全场把剑往地上一插——"我不服。你等着，我会赢回来的。"然后就跟上他了，甩都甩不掉 |
| 微妙转变 | 第300章 | 楚渊中了埋伏，重伤昏迷。她一路背着他杀出重围，三天三夜没合眼。他醒来时她立马甩开他，冷着脸说"别误会，只是不想你死在别人手上。" |
| 生死相托 | 第380章 | 被仇家围杀至绝境，两人背靠着背，浑身是血。她忽然笑了一声——"喂，要是今天死在这儿了，我有个秘密还没告诉你。"楚渊问什么秘密，她说"不告诉你，活着再说。" |
| 第一次破防 | 第450章 | 九天十地绝境，两人并肩死战三天三夜。她力竭倒下，被他一把拽进怀里挡住致命一击。她靠在他胸口听着他的心跳，忽然就哭了——那是她这辈子第一次在别人面前哭。"你要是敢死在这里，我追到黄泉也要把你拽回来。" |
| 误会与裂痕 | 第520章 | 姬家不同意她和楚渊来往，给她订了一门古族联姻。她没有告诉楚渊，一个人回去处理。楚渊从别人口中得知，赶到姬家时，她正在大堂上当着全族的面把婚书撕了。他对她说"你可以跟我说的。"她冷笑着反问："说了又如何？你是能替我摆平一个古族，还是能带我走？"两人不欢而散 |
| 破镜重圆 | 第580章 | 楚渊在生死关头得知她独自潜入敌营刺杀邪王，不顾一切追去。两人联手击杀邪王后，她在血泊中靠着他的肩，哑着嗓子说——"我以为你不会来了。""我什么时候没来过？"她没再说话，但她的手，悄悄握住了他的衣角 |
| 最痛的牺牲 | 第920章 | 域外邪族总攻，她奉命守住虚空山最后一道防线。楚渊在另一条战线上被拖住，赶到时，她已战至道渊崩碎，浑身浴血。她靠在残垣断壁上，看到他来了，第一句话是——"看，我守住了。"第二句话是——"别哭，你知道我不喜欢看你哭。" |
| 告别 | 第990章 | 最终决战前，她把破军剑横在楚渊面前，说——"你走你的道渊尽头。我在这里，等你回来。"他没说话，握住了她持剑的手。她用尽全力回握了一下，然后松开——"去吧。我会替你守好这个世界的。" |

**暧昧速写举例：**
> 楚渊的伤口还在渗血，姬墨音面无表情地撕下自己的衣袖，粗暴地替他包扎。下手很重，疼得他倒吸一口凉气。
> "你就不能轻点？"
> "嫌疼就别受伤。"她嘴上这么说，手上的力道却不由自主地轻了三分。
> 包好之后她没有松手，低头看着那个丑得不像话的结，忽然说了一句："下次再有这种事，叫上我。"
> "你不是说不管我死活吗？"
> "我反悔了。"她抬起头，目光直直地撞进他的眼睛里——"你有意见？"
> 楚渊看着那双明亮又倔强的眼睛，忽然笑了。
> "没有。"
> 她哼了一声，转过头去。但他分明看到，她的耳尖，红得像要滴血。

## 关于双女主感情线的创作思路

**紫瑶线 → 糖（顺滑暧昧）**
- 宿命的牵引让他们的相遇自带暧昧底色
- 感情发展顺滑自然，中间没有狗血误会
- 暧昧的张力来自于"清冷之人的破格之举"——她越矜持，偶尔一次主动就越撩人
- 这条线是全书的情感稳定器——不管经历了多少风雨，回头一看，她还在那里

**姬墨音线 → 虐（曲折意外）**
- 从冤家开始，刀子嘴豆腐心，嘴上不饶人
- 感情发展被身份、家族、使命一次次打断
- 暧昧的张力来自于"嘴硬之人的身体诚实"——她越否认，行动就越出卖她
- 这条线是全书的情感催化剂——每一次她的破防，都是楚渊的成长

**两人之间的关系：**
- 不是情敌，不是撕逼
- 紫瑶在楚渊离开青云域之前就知道有姬墨音这个人——因为她在他的命格里看到过"另一团火"
- 姬墨音第一次见紫瑶时，拔剑说"听说你是他很重要的人"，紫瑶平静地回答"你也是他很重要的人。"
- 两人交手一次，不分胜负。后来紫瑶救过姬墨音的命，姬墨音也为紫瑶挡过刀
- **她们对彼此的定位是：** "那个让他变得更好的人"
