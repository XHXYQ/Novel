# 《道渊》完整人物设定

---

## 主角 · 楚渊（唯一男主）

**基础设定：**
- 青云城楚家旁支孤儿，父母早逝
- 真实身份：太古神魔遗孤，血脉被人为封印后遗弃在低级世界
- 命格：九印之体——万古唯一能承载荒帝完整道渊的容器

**人品底色：**
楚渊骨子里是一个"被逼出来的守护者"。他不渴望权力，不迷恋力量。他变强的原因很简单——不想再看到同族死在异族的手里。他最大的恐惧不是死，是"保护不了想保护的人"。

**性格成长弧：**
```
100章前：隐忍求生，眼中只有自己的生存
  → 200章：看到人族苦难，开始有了"想守护什么"的念头
    → 400章：从"想守护几个人"到"想守护一个种族"
      → 650章：失去挚友后浴火重生
        → 1000章：化为道渊，守护诸天
```

**自创功法：无名炼体诀（全书暗线）**

楚渊没有师父手把手教他修炼。吞天诀是荒帝给的传承，但如何运用这股力量、如何保护自己不被人杀死——这是他用自己的命换来的领悟。

**诞生逻辑：**
吞天诀的核心是"吞噬外力化为己用"，但楚渊的经脉在一次次吞噬中不断被撑裂、修复、再撑裂。在一次濒死的边缘，他本能地将吞噬来的真气不是炼化为己用，而是打散后融入自己的血肉骨骼中——他发现自己变硬了。

**从此，一条独属于楚渊的修炼之路开始了。**

**功法特点：**
- 以吞天诀吞噬的外来能量为燃料，淬炼肉身
- 每一次濒死都是一次突破——打不死的只会让他更硬
- 初期粗糙简陋，只是被动防御；中期可控强化，主动锻体；后期演化出独立体系
- 不依赖灵气、不依赖天赋——只依赖生死战斗

**进化路线：**
| 阶段 | 章节 | 描述 |
|------|------|------|
| 萌芽 | 约第50章 | 第一次濒死时本能将吞噬能量融入血肉，意外扛住致命一击 |
| 雏形 | 约第150章 | 开始有意识地用此法锻体，无名无姓，只称"那套法子" |
| 初成 | 约第250章 | 姬墨音重剑劈在他身上只留下一道白痕，她第一次露出震惊的表情 |
| 命名 | 约第500章 | 在一次独战万军的战役后，幸存者问他这是什么功法，他说"没名字"。从此世人称其为——**不灭道体** |
| 大成 | 约第750章 | 可硬抗至尊一击，肉身堪比圣器 |
| 极致 | 约第950章 | 万法不侵，万古不灭——连道渊尽头的力量也无法摧毁他的肉身 |

**与吞天诀的关系：**
- 吞天诀是刀——锋利可伤人，但刀本身也会断
- 炼体诀是刀鞘和铸刀的铁砧——让这把刀永远不会折断
- 两者同源：吞天诀吞噬能量，炼体诀用这些能量锻造身体
- 两者相辅相成：没有吞天诀提供能量，炼体诀进展缓慢；没有炼体诀锻造的身体，吞天诀会把楚渊自己先撑爆

**核心矛盾：**
1. 吞天诀越强，戾气越重——每一次突破都在靠近"非人"
2. 九印越完整，寿命越短——变强的代价是活不长
3. 炼体诀越强，他越不像一个"普通的修士"——当他肉身不灭时，他还是"人"吗？
4. 他越是想守护身边的人，身边的人就越容易因为他而死
5. 他知道自己终有一天要走到道渊尽头——他不敢给任何人承诺，怕自己做不到

## 女主一 · 紫瑶

**登场：第100章末**

**基础设定：**
- 青云宗圣女，常年闭关，极少露面
- 姓"紫"，名"瑶"，紫衣如烟，清冷出尘
- 真实身份：荒帝直系后裔，体内封印远古天凤血脉
- 修为：出场即为神桥境，远超同期弟子

**初遇场景（第100章）：**
妖族统领率军入侵青云宗，人族防线即将崩溃之际——一道紫光从后山冲天而起。紫瑶一袭紫衣踏空而来，一掌震退妖族统领，救了濒死的楚渊。两人在空中对视，楚渊感到道渊深处的九道裂痕同时震动了一瞬。紫瑶收回目光，面无表情地说了一句："愣着干什么？还站得起来吗？"

**性格：**
外冷内热，表面清冷疏离，实则极为在乎身边的人。从不说肉麻的话，但每一个行动都在乎。她的爱不是轰轰烈烈的告白，而是"在你睡着时替你守夜/\f在你受伤时偷偷往你房里送药/\f在你离开时追到城外塞给你一件护身符"。

**感情线（顺滑暧昧）：**
第100章 初遇→第120章 战后清理中建立信任→第150章 月下谈心→第250章 八荒重逢→第300章 暧昧升温→第400章 天墟定情→第830章 为救楚渊陨落→第832章 被时印救回→第980章 并肩终战

**纯爱保障：**
她从第100章初遇起，心里就再也装不下别人。清冷之人不动心则已，一动心就是一生。

## 女主二 · 姬墨音

**登场：第200章**

**基础设定：**
- 战皇殿圣女，古族姬家继承人
- 姓"姬"，名"墨音"——古族的古老姓氏，是荣耀也是枷锁
- 武器：黑色重剑"破军"，八千斤
- 性格：锋芒毕露、从不服软、刀子嘴豆腐心
- 修为：登场时与楚渊同级（化海境），是他在八荒遇到的第一个旗鼓相当的对手

**初遇场景（第200章）：**
八荒排名大会上，楚渊连胜十二场后，一座擂台被人从中间一剑劈开。烟尘中，姬墨音扛着比她人还高的重剑走进来，目光扫了一圈，落在他身上——"听说你很能打？来，打一场。"

两人打了三天三夜，从擂台打到荒野，从荒野打到山谷，最后双双力竭倒在溪水里。她躺在水里仰天大笑——"痛快！你叫什么名字？"
"楚渊。"
"我叫姬墨音。记住这个名字——下次见面，我会赢你。"

**性格：**
表面冷傲霸道、杀伐果断、从不示弱。内心极度柔软，但不善于表达——她表达在乎的方式是"嘴上说着狠话/\f身体却无比诚实"。她会骂楚渊"你逞什么能"，然后挡在他身前替他挨一刀。她会在众人面前说"谁管你的死活"，然后在他重伤昏迷时装作路过，守了整整一夜。

**感情线（曲折虐心）：**
第200章 冤家初遇，打了三天三夜→第250章 排名大会再战，她输半招，赖上他了→第300章 微妙转变（背他杀出重围，三天三夜没合眼）→第380章 第一次破防（血泊中哭着说"你要是敢死在这里，我追到黄泉也要把你拽回来"）→第520章 家族阻挠（姬家不同意，她当面撕了婚书）→第580章 破镜重圆（生死关头重逢，她握住他的衣角）→第920章 战至道渊崩碎（"看，我守住了"）→第990章 最终告别（"你走你的道渊尽头。我在这里，等你回来。"）

**虐的来源：**
- 她的骄傲——她从不低头、从不解释，所以误会只能靠时间和生死来消解
- 她的家族——古族姬家不可能接受一个人族女婿，这是她与楚渊之间最大的外部阻挠
- 她的不善表达——她心里有十分，嘴上只说得出一分；而楚渊有时候也笨，看不出那一分背后的九分

**纯爱保障：**
她虽然嘴上不承认、路上波折多，但她对楚渊的感情从未动摇过。撕婚书不是因为她生气——是因为她从来没想过要嫁给别人。她的虐点全部来自外部，没有任何变心或背叛。

## 双女主关系定位

**两人不是情敌。**
紫瑶在楚渊的命格里看到过"另一团火"。姬墨音第一次见紫瑶时拔剑说"听说你是他很重要的人"——紫瑶平静地回答"你也是。"
两人交手一次，不分胜负。后来紫瑶救过姬墨音的命，姬墨音也为紫瑶挡过刀。
她们对彼此的定位是——"那个让他变得更好的人。"

**楚渊对两人的感情：**
对紫瑶是灵魂的归宿——他在她面前可以卸下所有防备。对姬墨音是最信任的后背——他可以把命交到她手上，也知道她会接住。
他从未对紫瑶隐瞒姬墨音，也从未对姬墨音隐瞒紫瑶。
他不敢承诺未来——因为他知道自己终究要去道渊尽头。但他用自己的命，守护了她们直到最后一刻。
