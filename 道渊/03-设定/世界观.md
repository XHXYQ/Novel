# 《道渊》世界观设定

## 世界格局

### 青云域
- 边陲小域，灵气稀薄，强者稀少
- 最强者仅为神桥境
- 楚渊的故乡，故事的起点

### 八荒世界
- 主舞台，浩瀚无垠
- 分为八荒：东荒、西漠、南岭、北原、中州、天墟、幽冥海、虚空山
- 圣地林立，古族盘踞
- 最强者为至尊境，大帝已成传说

### 九天之上
- 传说中超越八荒的世界
- 远古大帝的陨落之地
- 荒帝与域外邪族的最终战场
- 有大帝机缘，亦有必死之局

### 虚空裂缝
- 域外邪族的封印之地
- 位于九天之上的尽头
- 封印由远古诸帝联手布置，如今已出现裂痕

### 道渊体系（核心创新）
- **道渊**：每个人体内的神秘空间，蕴含灵魂本源
- **觉醒**：12-18岁之间道渊会自然觉醒，觉醒后才有资格修行
- **品质**：从低到高分为灰、青、蓝、紫、金、彩色六等
- **碎裂道渊**：楚渊的道渊因继承荒帝碎片而呈碎裂状，在任何人看来都是废品
- **道渊技**：每个道渊自带独特能力，楚渊的道渊能力是"吞噬"——可吞噬天地万物化为己用

### 功法等级
凡阶 → 灵阶 → 玄阶 → 地阶 → 天阶 → 圣阶 → 帝阶

## 宝物设定

### 天材地宝
- **灵气浓度**：普通 < 浓郁 < 宝地 < 洞天 < 福地 < 圣地
- **灵药等级**：十年 < 百年 < 千年 < 万年 < 十万年 < 神药

### 兵器等级
凡器 → 灵器 → 宝器 → 道器 → 圣器 → 帝兵

## 种族设定
- **人族**：主流种族，占据八荒世界
- **古族**：远古血脉传承，拥有独特天赋
- **妖族**：妖兽修炼化形，占据妖域
- **域外邪族**：来自天外的邪恶种族，全书最终BOSS
